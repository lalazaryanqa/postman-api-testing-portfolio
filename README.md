# Postman API Testing Portfolio Project

This project demonstrates automated API testing using Postman and JSONPlaceholder REST API.

The collection includes CRUD operations, environment variables, request chaining, automated tests, collection runner execution, and negative testing.

---

# Technologies Used

- Postman
- REST API
- JSON
- JavaScript
- Collection Runner
- Environment Variables

---

# Features

## CRUD Operations

- GET all posts
- GET single post
- POST create post
- PUT update post
- PATCH update title
- DELETE post

---

## Automated Testing

The project includes automated tests for:

- Status code validation
- Response validation
- JSON parsing
- Dynamic variables
- Request chaining
- Negative testing

# Negative Testing
